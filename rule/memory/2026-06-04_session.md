## 完成事项

### 1. /resume 会话管理命令
- **语法**: `/resume [list|list all|clear|delete <id>|import <file>|<id>]`
- **list**: 列出所有会话（ID、时间、消息数、LLM标题），用第一条用户消息作为fallback标题
- **list all**: 额外展开每个会话前2条用户消息
- **clear**: 清空所有会话记录，自动创建新会话
- **delete <id>**: 删除单个会话及其消息
- **import <file>**: 导入 /export 导出的 .md 文件，解析消息创建新会话
- **<id>**: 切换到指定会话，加载历史消息
- **涉及文件**: `cli.py`（_handle_resume ~50行）、`session/store.py`（+delete_conversation, +delete_all_conversations）、`ui/input_handler.py`（SLASH_COMMANDS + SUBCOMMANDS）

### 2. LLM自动标题
- 每个会话首轮对话后，调LLM用5-10字总结标题，存入 conversations.summary
- **涉及文件**: `cli.py`（_generate_title方法）、`session/store.py`（+update_summary）

### 3. 延迟创建会话（根除0msgs空记录）
- run()不再立即create_conversation，改为首条用户消息发出时才写入DB
- 新增 _conv_created 标记追踪
- **涉及文件**: `cli.py`

### 4. /export 导出会话
- `/export [id]` 导出为 Markdown 文件，含完整元信息和对话记录
- 文件命名: `TUDOU_export_<标题>_<时间>.md`
- **涉及文件**: `cli.py`（_handle_export方法）、`ui/input_handler.py`

### 5. 对话内任务面板
- **三个工具**: TaskCreate、TaskUpdate、TaskList — agent可在对话中调用
- **一个命令**: `/tasks` — 用户手动查看
- **实时渲染**: 每次工具调用后自动刷新面板（有活跃任务时），全部完成自动隐藏
- **视觉效果**: 波浪渐变色标题 + 树状图布局 + 已完成项strikethrough划线
- **持久化**: `~/.tudou_agent/tasks_<conv_id>.json`
- **涉及文件**: `tools/task_manager.py`（新建）、`cli.py`、`context/manager.py`（系统提示词）、`ui/input_handler.py`
