## 完成事项 (3大功能 + 版本升级)

### 1. 启动速度优化 — 技能懒加载 + JSON 索引缓存
- `skills/loader.py`: Skill 新增 `_body_loaded` + `_frontmatter_raw` 字段，`ensure_body()` 触发延迟加载；`SkillLoader` 新增 `load_stub()`（只解析 frontmatter 跳过 body）和 `_load_body()`
- `skills/registry.py`: 新增 JSON 缓存（`~/.tudou_agent/skill_index.json`），`discover()` 优先读缓存（mtime 比对），缓存不新鲜时扫描但只做 stub 解析，解析完自动写缓存；新增 `invalidate_cache()` 和 `_save_to_cache()`
- `cli.py`: `_import_danger_skill`/`_remove_skill`/`_handle_skills_install` 中 skill 变更后刷新缓存

### 2. 终端标题栏动态更新

- 调用点: `__init__` 末尾、`/model`、`/resume` 各分支（auto-resume/import/clear/切换会话）

### 3. 多模态支持 — 图片/PDF 读取
- `tools/base.py`: ToolResult 新增 `images: list[dict] | None` 字段
- `tools/file_read.py`: 完全重写，图片(PNG/JPG/GIF/WebP/BMP)→base64编码，PDF→pypdf 提取文本，20MB 上限
- `agent.py`: `_append_tool_messages` 检测 `result.images`，构建 `[{type:text}, {type:image, source:{...}}]` 数组 content
- `llm/providers/anthropic.py`: 新增 `_convert_content()` 处理列表 content，`_convert_message()` 支持列表 content
- `llm/providers/openai_compat.py`: 新增 `_convert_messages()` + `_convert_content()`，将图片块转为 OpenAI `image_url` 格式（data URI）
- `requirements.txt`: 新增 `pypdf>=4.0.0`
